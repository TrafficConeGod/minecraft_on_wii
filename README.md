# WiiCraft

A Minecraft-like clone for the Nintendo Wii 

### Build instructions:

#### Prerequistites

 - devkitPPC + portlibs
 - devkitPro tools (in $PATH for gxtexconv)

#### Building

 - Run make in the repository root
